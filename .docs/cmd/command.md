# CMD команды

Обмен пакетами с ресурсом

```
ping <адрес>
```

Трассировка маршрута к ресурсу

```
trcert <адрес>
```

Узнать адрес ресурса

```
nslookup <адрес>
```

Настройки протокола IP для Windows

```
ipconfig
```

DNS записи

```
ipconfig /displaydns
```

Очистить кэш сопоставителя DNS

```
ipconfig /flushdns
```

Активные подключения

```
netstat -ano
```
